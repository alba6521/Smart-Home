# Smart-Home

## 🎯 Objetivos

* Diseñar una **Landing Page de un Dashboard de Control para una Smart Home**.
* Crear una interfaz visual con **widgets informativos** (temperatura, luces, consumo energético y cámaras).
* Implementar un **modo día y modo noche** utilizando exclusivamente **variables CSS**.
* Construir un diseño **responsive** que funcione tanto en **móvil** como en **tablet o escritorio**.
* Aplicar la metodología **Mobile First**, desarrollando primero la versión móvil y adaptándola después a pantallas mayores.

---

## 🛠️ Tecnologías

* **HTML5**
* **CSS**
* **Figma**

---

## 📋 Requisitos del Proyecto

* No se permite el uso de **px** 
* Implementar un **sistema de variables CSS con capas de abstracción**
